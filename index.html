Componentes de Bootstrap

Accordion

Es un componente que organiza el contenido dentro de elementos que se contraen.

Ejemplo Accordion

Alerts

Son mensajes de alerta para acciones que realizan nos usuarios.

Ejemplo Alerts

Notifications

Badge: se utilizan para agregar información adicional a cualquier contenido.

Spinners: indican el estado de carga de un componente o página con giradores.

Ejemplo Badge Ejemplo Spinners

Information

Breadcrumb: se utiliza para darle sentido a una página web y sirven para orientar el camino recorrido al usuario que navega por la misma.

Pagination: documentación y ejemplos para mostrar la paginación para indicar que existe una serie de contenido relacionado en varias páginas.

Ejemplo Breadcrumb Ejemplo Pagination

Buttons

Buttons: se utilizan para acciones en formularios, cuadros de diálogo y más con soporte para múltiples tamaños, estados y más.

Button group: agrupa una serie de botones en una sola línea o apila en una columna vertical.

Close button: botón de cierre genérico para descartar contenido como modales y alertas.

Ejemplo Buttons Ejemplo Button group Ejemplo Close button

Cards

Es una caja con bordes con algo de relleno alrededor de su contenido. Incluye opciones para encabezados, pies de página, contenido, colores, etc.

Ejemplo Cards

Carousel

Presenta diapositivas para recorrer elementos (imágenes o diapositivas de texto) como un carrusel.

Ejemplo Carousel

Collapse

Indica un elemento plegable; este es el contenido que se muestra o se oculta con un clic de un botón.

Ejemplo Collapse

Dropdowns

Es una lista desplegable es una interfaz visual que al ser presionada muestra una lista de opciones que puede elegir el visitante.

Ejemplo Dropdowns

List group

Los grupos de listas son un componente flexible para mostrar una serie de contenido.

Ejemplo List group

Modal

Se utiliza para agregar cuadros de diálogo a su sitio, notificaciones de usuario o contenido completamente personalizado.

Ejemplo Modal

Navs & tabs

Son pestañas que sirven para navegar dentro del sitio.

Ejemplo Navs & tabs

Navbar

La barra de navegación que se ubica generalmente en la parte superior de la página contiene los enlaces principales para navegar el sitio web.

Ejemplo Navbar

Offcanvas

Es un componente de la barra lateral que se puede alternar para que aparezca desde el borde izquierdo, derecho, superior o inferior de la ventana del navegador.

Ejemplo Offcanvas

Placeholders

Se utilizan marcadores de posición de carga para indicar que es posible que todavía se esté cargando algo.

Ejemplo Placeholders

Popovers

Es un pluggin similar a la información de herramientas; se trata de un cuadro emergente que aparece cuando el usuario hace clic en un elemento.

Ejemplo Popovers

Progress

Son barras que permiten mostrar de forma continua el estado de ejecución de una tarea.

Ejemplo Progress

Scrollspy

Se utiliza para actualizar automáticamente los enlaces en una lista de navegación basada en la posición de desplazamiento.

Ejemplo Scrollspy

Toasts

Es como un cuadro de alerta que solo se muestra durante un par de segundos cuando ocurre algo (es decir, cuando el usuario hace clic en un botón, envía un formulario, etc.).

Ejemplo Toasts

Tooltips

Es pequeño cuadro emergente que aparece cuando el usuario mueve el puntero del mouse sobre un elemento.

Ejemplo Tooltips

Tables

Se utilizan para representar datos en filas y columnas dentro de una página web.

Ejemplo Tables

Forms

Se utilizan para que el usuario introduzca datos que le peden y son enviados a un servidor para ser procesados.

Ejemplo Forms
